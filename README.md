# 汽车领域 RAG 知识问答系统
本项目为个人技术练习项目，无任何商业用途，是依托本地 PDF 资料搭建的检索增强生成问答应用。项目参考汽车产品使用说明类文档作为数据源，可解析 PDF 内文字与图片资源、构建本地化向量索引，搭配前端 Web 页面实现中文智能问答。
系统收到用户提问后，通过向量检索匹配文档关联片段，结合大模型生成答复，同时展示答案对应的原文出处、页码、引用段落及相似度数值，便于溯源核验回答内容。

## 功能特点

- 支持本地 PDF 知识库索引构建
- 支持文本内容切分、向量化和本地检索
- 支持 PDF 图片内容提取与视觉描述入库
- 支持中文自然语言问答
- 支持上传图片辅助提问，例如仪表盘图标识别
- 回答结果展示来源页码和引用片段
- 提供 FastAPI 后端接口和本地 Web 聊天界面
