# 问界 M6 说明书 RAG 知识问答系统

这是一个基于本地 PDF 文档构建的 RAG 知识问答项目。项目以《问界 M6 使用说明书》为知识源，支持从 PDF 中提取文本和图片内容，构建本地向量索引，并通过 Web 页面进行中文问答。

系统会根据用户问题检索相关说明书片段，调用大模型生成回答，并展示来源文档、页码、相关片段和相似度，方便用户验证答案依据。

## 功能特点

- 支持本地 PDF 知识库索引构建
- 支持文本内容切分、向量化和本地检索
- 支持 PDF 图片内容提取与视觉描述入库
- 支持中文自然语言问答
- 支持上传图片辅助提问，例如仪表盘图标识别
- 回答结果展示来源页码和引用片段
- 提供 FastAPI 后端接口和本地 Web 聊天界面
